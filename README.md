# Real-Estate-Agency

This is a real estate website built using HTML, CSS, and Bootstrap. The website is fully responsive and mobile-friendly.

## Getting Started

To get started, clone the repository to your local machine.

```
git clone https://github.com/your-username/Real-Estate-Agency.git
```

Once the repository is cloned, open the `index.html` file in a text editor. This is the main HTML file for the website.

## HTML

The HTML code for the website is fairly straightforward. The `<header>` section contains the website's logo and navigation bar. The `<main>` section contains the main content of the website, such as the property listings and contact information. The `<footer>` section contains the website's copyright information.

## CSS

The CSS code for the website is used to style the HTML elements. The CSS code is located in the `style.css` file.

## Bootstrap

Bootstrap is a CSS framework that is used to create responsive websites. The Bootstrap framework is included in the `bootstrap` folder.

## Running the Website

To run the website, open the `index.html` file in a web browser. The website should display correctly in all major web browsers.

## Conclusion

This is a simple real estate website that was built using HTML, CSS, and Bootstrap. The website is fully responsive and mobile-friendly.
